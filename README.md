# 这是一个Beamer模板

本模板主要基于 [Oxford Mathematical Institute Beamer Presentation](https://cn.overleaf.com/latex/templates/oxford-mathematical-institute-beamer-presentation/zxkbrjpszxjr) 的样式

修改了配色、校徽和封面背景，并添加了顶部导航条。

模板中的文字内容来自[THU-Beamer](https://github.com/Trinkle23897/THU-Beamer-Theme)，字体设置为宋体。

如果需要更换校徽、院徽和封面背景，请替换images文件夹中的图片文件，并修改*beamerthemezuel.sty*文件中的路径。
